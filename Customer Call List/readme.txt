## Customer Call List 

This folder contains three files:
Customer Call List.xlsx - original dataset (excel worksheet)
Cleaned Customer Call List.xlsx - cleaned dataset (excel worksheet)
CustomerCall_List.ipynb - python code used to clean dataset (jupyter notebook code file) 

The dataset has been cleaned as per the following requirement: 
Provide clean data only for customers who can be contacted. 

Overview: 
The cleaning of this project has been done using pandas module.
